Pong Game

Shortly after my graduation from Tech360, I thought it would be nice to learn some things about other coding languages (guess it would be more appropriate to say 'programming languages'). With that thought in mind, I decided to learn some basic stuff in Python from YouTube videos (I haven't learned everything in Python) and after some messing around with Python, I decided to try and make a simple and easy game: Pong. Initially, I tried to do this in Pygame, but I gave up on that quickly because it was a bit complicated to understand the functions/commands pygame uses. Instead, I used turtle to make the game and with the help of some YouTube videos to understand turtle functions, I had made the game. This was done in Pycharm first though, so there are some scrapped things in this repl.it because I can't import Winsound for music. Repl.it really does downgrade the quality of the game though.
